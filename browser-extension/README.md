## Responsively Helper Browser Extension
A nifty browser extension developed by the Responsively community to conveniently open the current browser link in the Responsively App!
<br><b>Requires Responsively already installed.</b> [Install intructions can be found here.](https://github.com/responsively-org/responsively-app/blob/main/README.md)
<br>Compatible with:</br>
- Chrome
- Firefox
- Microsoft Edge

## Features
- Quickly open your current browser URL in the Responsively app
- Test the responsiveness of the website you're currently browsing across multiple different devices
- Mimic the interactions with the website you're currently browsing across multiple different devices

## Issues

If you face any problems while using the application, please open an issue here - https://github.com/responsively-org/responsively-app/issues

## Contribute

1. Ensure you have installed `node` and `yarn`
2. Go to `browser-extension` folder
3. Run `yarn` to install dependencies
4. Run `yarn dev`. This will start the app for local development with live reloading.
